System

The Exposure Triangle is the canonical photographic model: three parameters (ISO, Aperture, Shutter Speed) must balance to achieve correct exposure. Unlike the Fire Triangle (synergistic), this is compensatory — adjusting one requires compensating another.

Reciprocity Law

EV = log₂(ISO) + log₂(1/f²) + log₂(t) = const Infinite combinations yield the same EV but different image qualities.

Three Parameters

ISO Sensor gain — sensitivity vs noise/grain Aperture f-number — light flux vs depth of field Shutter Exposure time — light vs motion blur

Trade-offs (m2:Trade-off)

Cannot simultaneously maximize: • DoF (deep focus) → narrow aperture (f/11+) • Freeze motion → fast shutter (1/1000s+) • Low noise → low ISO (100–400)

EV Gauge (left)

Shows signed offset from daylight baseline (ISO 400, f/5.6, 1/250s): 0 = correctly exposed · − = under · + = over

Analogy to Fire Triangle

Both use 3-component triangular models — but Fire Triangle is synergistic (all must co-exist), Exposure Triangle is compensatory (any one can compensate another). Same geometry, different logic.
