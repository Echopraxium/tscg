· VCO — Voltage Controlled Oscillator
Electronics / Modular Synth m3:Poclet v1.0.0

System

A VCO (Voltage Controlled Oscillator) is the fundamental pitch-generation component of a modular synthesizer. It produces a periodic audio-frequency waveform whose frequency is precisely controlled by an input voltage following the V/Oct standard (1 volt per octave).

V/Oct Law

f(CV) = f₀ × 2^(CV / 1V) At f₀ = 440 Hz (A4): CV = 0V → 440 Hz (A4) CV = +1V → 880 Hz (A5, +1 octave) CV = -1V → 220 Hz (A3, -1 octave) 1/12 V ≈ 83 mV → +1 semitone

Waveform Modes

Sine Fundamental only. Pure, flute-like. Square Odd harmonics (1/n). Hollow, woodwind. Triangle Odd harmonics (1/n²). Soft, mellow. Sawtooth All harmonics (1/n). Bright, string/brass.
