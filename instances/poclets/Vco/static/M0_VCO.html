System

A VCO (Voltage Controlled Oscillator) is the fundamental pitch-generation component of a modular synthesizer. It produces a periodic audio-frequency waveform whose frequency is controlled by an input voltage via the V/Oct standard.

V/Oct Law

f(CV) = f₀ × 2^(CV / 1V)   CV = 0V → 440 Hz (A4)  |  +1V → 880 Hz (+1 oct)   CV = −1V → 220 Hz (−1 oct)  |  1/12 V → +1 semitone

Waveform Modes

Sine — Fundamental only — pure, flute-like Square — Odd harmonics (1/n) — hollow, woodwind Triangle — Odd harmonics (1/n²) — soft, mellow Sawtooth — All harmonics (1/n) — bright, string/brass

V/Oct Exponential Law
f(CV) = f₀ × 2^(CV / 1V)
This is an m2:Amplification in the log domain: each +1V doubles frequency (one octave). Musical intervals are therefore linear in CV space, enabling transposition by simple voltage addition.
Mode Switching — m2:Mode

Each mode selects a different harmonic profile (Information dimension I): Sine=pure, Square=odd 1/n, Triangle=odd 1/n², Sawtooth=all 1/n. Same pitch, different timbre — orthogonal axes of the sound design space.
