# TRIZ in a Nutshell: The 40 Inventive Principles

## What is TRIZ?

**TRIZ** (Russian acronym for *Theory of Inventive Problem Solving*) is a problem-solving and innovation methodology developed by Soviet inventor **Genrich Altshuller**.

By analyzing hundreds of thousands of patents, Altshuller discovered that while technical problems seem unique, **truly inventive solutions follow repeating patterns**. TRIZ provides a systematic way to break free from "psychological inertia" (trial-and-error) by applying these universal solution models.

## The Core Concept: Contradictions

At the heart of TRIZ is the idea that a difficult problem often stems from a **contradiction**. There are two main types:

1.  **Technical Contradiction:** A classic trade-off. Improving one feature (e.g., engine power) worsens another (e.g., fuel consumption).
2.  **Physical Contradiction:** One element must have two opposite properties simultaneously (e.g., a glass must be both **transparent** to see through and **opaque** to block light).

TRIZ offers two main tools to resolve contradictions:
- **The Contradiction Matrix:** Cross-references 39 technical parameters to suggest relevant principles.
- **The 40 Inventive Principles:** The universal solution patterns listed below.

## The 40 Inventive Principles (Summary with Examples)

| # | Principle Name | Concise Description | Example |
| :--- | :--- | :--- | :--- |
| **1** | **Segmentation** | Divide an object into independent parts. | Replace a mainframe computer with a network of PCs. |
| **2** | **Extraction** | Separate a "disturbing" part, or extract only the needed part. | Place a noisy air compressor outside the building. |
| **3** | **Local Quality** | Change a structure from uniform to non-uniform. | A pencil with an eraser, or a hammer with a claw. |
| **4** | **Asymmetry** | Replace a symmetrical form with an asymmetrical one. | Add a flat side to a cylindrical shaft for a secure grip. |
| **5** | **Merging** | Bring identical or related objects together. | Perform multiple medical tests on a single blood sample. |
| **6** | **Universality** | Make an object perform multiple functions. | A toothbrush handle that contains toothpaste. |
| **7** | "Nesting" (Russian Dolls) | Fit one object inside another. | A telescopic radio antenna, or Russian dolls. |
| **8** | **Anti-weight** | Compensate weight by combining with another object that provides lift. | A helium balloon to support an advertising sign. |
| **9** | **Preliminary Anti-action** | Apply a pre-stress (often opposite) to an object. | Pre-stressing steel bars before pouring concrete. |
| **10** | **Preliminary Action** | Perform an action in advance. | Pre-glued wallpaper. |
| **11** | **Beforehand Cushioning** | Prepare emergency means to counter low reliability. | A backup parachute. |
| **12** | **Equipotentiality** | Change working conditions to avoid lifting/lowering objects. | The Panama Canal (uses locks to change water level). |
| **13** | **Do it in Reverse** | Invert an action, or make a fixed part movable. | To free a jammed part, cool the inner part (instead of heating the outer one). |
| **14** | **Spheroidality** | Replace linear parts with curved ones, planes with spheres. | Using ball casters to move furniture. |
| **15** | **Dynamics** | Allow an object to change for optimal performance. | An adjustable car steering wheel or seat. |
| **16** | **Partial/Excessive Action** | If 100% is hard to achieve, do "a little less" or "a little more." | Applying an extra layer of paint, then removing the excess. |
| **17** | **Another Dimension** | Move from 1D to 2D, 2D to 3D, or use the opposite side. | Double-sided electronic circuit boards, stacked chips. |
| **18** | **Mechanical Vibration** | Make an object vibrate (including ultrasound). | An electric knife with vibrating blades. |
| **19** | **Periodic Action** | Replace continuous action with periodic or pulsating action. | A pulsed siren (more alarming than a continuous sound). |
| **20** | **Continuity of Useful Action** | Keep an object's parts working continuously without idle time. | A printer prints during the return stroke of the carriage. |
| **21** | **Rushing Through** | Perform a harmful action at high speed to avoid damage. | A paper cutter for sharp-edged products. |
| **22** | **Convert Harm into Benefit** | Use harmful factors (e.g., waste heat) to achieve a positive effect. | Using waste heat from a factory to warm offices. |
| **23** | **Feedback** | Introduce feedback to improve a process. | A thermostat that turns a heater on/off. |
| **24** | **Intermediary** | Use an intermediary object to transfer or perform an action. | A carrier pigeon carrying a message. |
| **25** | **Self-service** | Make an object serve itself or perform its own maintenance. | A self-cleaning oven. |
| **26** | **Copying** | Use simple, inexpensive copies instead of fragile originals. | A virtual car crash simulation instead of a real one. |
| **27** | **Cheap Short-lived Objects** | Replace expensive, durable objects with cheap, disposable ones. | Disposable paper surgical masks. |
| **28** | **Replacement of a Mechanical System** | Use fields (e.g., magnetic, acoustic) instead of mechanical ones. | A magnetic levitation (Maglev) train. |
| **29** | **Pneumatics or Hydraulics** | Replace solid parts with gases or liquids. | An air or hydraulic suspension system. |
| **30** | **Flexible Shells or Thin Films** | Use flexible shells or thin films instead of bulky structures. | An inflatable mattress or bubble wrap. |
| **31** | **Porous Materials** | Make an object porous or add porous elements. | A metal sponge for filtering. |
| **32** | **Color Changes** | Change the color or transparency of an object. | Photochromic sunglasses. |
| **33** | **Homogeneity** | Make objects interact with the same material (or one with identical properties). | Ice cubes made from the same drink to avoid dilution. |
| **34** | **Discarding and Recovering** | Discard parts of an object (or let them dissolve) after use. | A self-dissolving medical stitch. |
| **35** | **Transformation of Properties** | Change the physical state (e.g., solid to liquid) of an object. | A classic "squib" fire extinguisher. |
| **36** | **Phase Transition** | Use phenomena occurring during a phase change (e.g., volume change, heat absorption). | A fridge uses the heat absorption of evaporating liquid. |
| **37** | **Thermal Expansion** | Use the expansion or contraction of materials with temperature change. | A bimetallic strip in a thermostat. |
| **38** | **Strong Oxidants** | Replace normal air with enriched or highly concentrated oxygen. | An oxyacetylene torch for cutting metal. |
| **39** | **Inert Atmosphere** | Replace a normal environment with an inert one to prevent unwanted reactions. | Using argon gas to prevent oxidation in a lightbulb. |
| **40** | **Composite Materials** | Replace a homogeneous material with a composite one. | Fiber-reinforced polymer (carbon fiber) for airplane parts. |

## Key Takeaway

**Instead of fighting a contradiction, use these principles to reframe and solve it.**

When you face a tough problem, identify the underlying contradiction. Then, look at the principles above. Which one could help you break the trade-off?
