## Design Insights

The Cluster/Hub dichotomy captures a fundamental systems distinction:
- Some structures **process** (Cluster = mini-brains)
- Some structures **relay** (Hub = plexuses)

This distinction appears everywhere:
- Biology: Ganglia (active) vs Plexus (passive)
- Urban: Innovation district (active) vs Train station (passive)
- Computing: Server cluster (active) vs Router (passive)

### 2. Analytical/Constructive Duality

- **Same structure** (SâŠ—I)

### 3. Compositional Hierarchy
Network decomposition shows clear levels:
1. Elementary: Node
