* install Python required modules:
pip install -r requirements.txt

* install "Google Cloud"
1. Open "Powershell" with _open_powershell.bat
2. type this in the powershell window:
   (New-Object Net.WebClient).DownloadFile("https://dl.google.com/dl/cloudsdk/channels/rapid/GoogleCloudSDKInstaller.exe", "$env:Temp\GoogleCloudSDKInstaller.exe") & $env:Temp\GoogleCloudSDKInstaller.exe

* Add "gcloud" to $PATH environment variable:
1. Open "Powershell" with _open_powershell.bat
2. type this in the powershell window:   
   $path = "$env:LOCALAPPDATA\Google\Cloud SDK\google-cloud-sdk\bin" [Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";$path", "User")
