M2 GenericConcepts Mobilized

This instance mobilizes 19 GenericConcepts from the M2 layer across 6 families (10 Structural, 4 Informational, 3 Ontological, 2 Regulatory).

● Structural (10)

Component S ⊗ I

Elementary constitutive part of a system with functional role in the whole. Distinguished from arbitrary partition by having meaningful function.

Role: 118 chemical elements as individual components

Hierarchy S ⊗ A

Nested levels of organization with vertical control relations.

Role: Multi-level organization (periods > groups > blocks > elements)

Modularity S ⊗ I

Decomposition into semi-independent functional units.

Role: Block structure (s, p, d, f) as modular units

Topology S ⊗ I

Classification of structural properties preserved under continuous deformations: connectivity, compactness, genus. Captures the TYPE or CLASS of structure independent of metric.

Role: 2D grid topology with lanthanide/actinide gaps

Symmetry S

Invariance under transformation.

Role: Periodic repetition (2n² shell capacity rule)

Segmentation S ⊗ I ⊗ D

Partition of a whole into homogeneous segments according to similarity, continuity, or structural criteria. Analytical decomposition producing distinct but exhaustive regions.

Role: Division into blocks by orbital type (l quantum number)

Node S ⊗ I

Elementary entity or point in a network structure, serving as connection endpoint.

Role: Each (period, group) position as a node

Network S ⊗ I ⊗ F

Set of nodes connected by edges enabling distributed interactions.

Role: Chemical family relationships (alkali metals, halogens, etc.)

Layer S ⊗ I ⊗ A ⊗ R

Stable, ordered positional partition in a stack: each layer has characteristic content and occupies a fixed rank defined by an attractor. No subordination (unlike Hierarchy): layers are co-equal strata.

Role: Electron shells as concentric layers (K, L, M, N, O, P, Q)

Identity S → I → A → V → E

Persistent property that makes an entity distinguishable and self-restoring across transformations. An entity remains itself despite internal changes.

Role: Unique identity per element (Z, symbol, name)

● Informational (4)

Code I ⊗ S

System of rules for systematic transformation between domains.

Role: Chemical symbols (H, He, Li...) as encoding system

Pattern S → I → A

Recurring structural motif identifiable across instances and reusable as a template. Exists as an observable regularity in Territory (Eagle Eye) and as a reusable archetype in Map (Sphinx Eye).

Role: Periodic trends (χ, r, IE gradients)

Signature I ⊗ S

Characteristic pattern enabling unique identification.

Role: Electron configuration as unique signature

State I

Instantaneous configuration of system variables.

Role: Oxidation states, phase states (gas/liquid/solid)

● Ontological (3)

Gradient ⊗₂F or ⊗₂I

Spatial or temporal variation creating driving force for flows.

Role: Property gradients (∇χ, ∇r, ∇IE)

Space S ⊗ I

Dimensional framework in which system states and trajectories exist (state space, phase space).

Role: Configuration space (n, l, m, s quantum numbers)

Resource F ⊗ I ⊗ R

A quantifiable entity available to, consumed by, or produced by a system. Characterized by Flow (F), Information (I), and Representability (R).

Role: Natural abundance, economic value of elements

● Regulatory (2)

Constraint S ⊗ I

Restriction on possible system states or trajectories, limiting degrees of freedom.

Role: Quantum rules (Pauli, Aufbau, Hund)

Threshold A ⊗ I

Critical value triggering qualitative change in system behavior.

Role: Noble gas configurations as stability thresholds
