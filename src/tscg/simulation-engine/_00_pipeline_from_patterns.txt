Utilité d'un Raisonneur dans votre pipeline
OUI, un raisonneur serait extrêmement pertinent pour plusieurs raisons :

1. Validation de cohérence ontologique
Vérification des contraintes entre concepts (ex: "une source de chaleur ne peut pas être une substance inflammable")

Détection des contradictions dans les relations

Inférence de nouvelles relations implicites

2. Vérification de la complétude
Identification des éléments manquants pour une simulation fonctionnelle

Suggestions d'ajouts basées sur des patterns reconnus

3. Optimisation des requêtes
Extraction intelligente des informations pertinentes pour la génération

Classification automatique des composants (entités, interactions, règles)

Avantages de cette approche :
Qualité garantie : Le raisonneur assure la cohérence avant génération

Réutilisabilité : Patterns validés pour des systèmes similaires

Scalabilité : La complexité est gérée par le raisonneur, pas par le LLM seul

Traçabilité : Liens clairs entre modèle conceptuel et code généré

class SimulationRAG:
    def __init__(self):
        self.vector_store = ChromaDB()
        self.reasoner = OWLReasoner()
        
    def generate_simulation(self, ontology_path):
        # 1. Validation ontologique
        validated_ontology = self.reasoner.validate(ontology_path)
        
        # 2. Extraction des concepts clés
        concepts = self.reasoner.extract_simulation_concepts()
        
        # 3. Recherche de patterns similaires
        similar_patterns = self.vector_store.query(concepts)
        
        # 4. Génération contextuelle
        return self.llm_generate(validated_ontology, similar_patterns)

    def __init__(self):

        self.embeddings = []

    def add_simulation(self, simulation_data):
        # Extraire les caractéristiques du système

def prepare_generation(nouveau_systeme):

# Architecture simple pour démarrer
class RAGSimulationPipeline:
    def __init__(self):
        self.pattern_db = SimulationPatternDB()
        self.reasoner = SimpleReasoner()
        
    def add_to_knowledge_base(self, simulation):
        # Extraction automatique
        pattern = self.extract_pattern(simulation)
        
        # Stockage vectoriel
        self.pattern_db.add(pattern)
        
        # Mise à jour des meta-patterns
        self.update_abstractions()
    
    def generate_new(self, ontology):
        # Recherche contextuelle
        similar = self.pattern_db.search(ontology)
        
        # Raisonnement sur l'ontologie
        validated = self.reasoner.validate(ontology)
        
        # Génération enrichie
        return self.generate(validated, similar)

## Overview

The **Fire Triangle** is the canonical poclet for TSCG framework validation. It represents the minimal yet complete system for combustion, demonstrating that fire requires exactly three components simultaneously: **Fuel**, **Oxygen (Oxidizer)**, and **Heat**. Removing any one component immediately stops the fire.

This poclet serves as:
1. **Framework Validation**: Proof-of-concept for TSCG's ASFID/REVOI analysis
2. **Metaconcept Discovery**: Helped identify Balance, Synergy, and Component metaconcepts
3. **Pedagogical Tool**: Entry point for teaching systems theory and fire science

## System Description

### The Three Components

#### 1. Fuel (Combustible)
- **Role**: Energy source
- **Function**: Provides chemical potential energy
- **Examples**: Wood, gasoline, natural gas, paper, coal
- **State**: Solid, liquid, or gas
- **Chemical Nature**: Typically hydrocarbons or organic matter
- **ASFID Contribution**:
  - **S** (Structure): Material structure of fuel
  - **F** (Flow): Fuel consumption flow
  - **I** (Information): Chemical composition information

#### 2. Oxygen (Oxidizer)
- **Role**: Electron acceptor
- **Function**: Enables oxidation reaction
- **Primary Example**: O₂ (atmospheric oxygen)
- **Alternative Oxidizers**: Cl₂, F₂, ClO₄⁻, NO₃⁻
- **Concentration**: Typically >16% in air for sustained combustion
- **ASFID Contribution**:
  - **F** (Flow): O₂ flow from environment
  - **D** (Dynamics): Reaction dynamics (oxidation rate)

#### 3. Heat (Activation Energy)
- **Role**: Activation threshold
- **Function**: Initiates and sustains reaction above ignition temperature
- **Sources**: Match, spark, friction, hot surface, lightning
- **Threshold**: T_ignition (fuel-dependent)
- **Sustainment**: Once ignited, combustion generates its own heat (positive feedback)
- **ASFID Contribution**:
  - **A** (Attractor): Thermal attractor (equilibrium temperature)
  - **D** (Dynamics): Thermal dynamics (heating/cooling)

### Synergy Principle

**Formula**: `Fuel ⊕ O₂ ⊕ Heat → Fire (emergent)`

- **Description**: All three components required **simultaneously**
- **Emergent Property**: Flame (self-sustaining exothermic reaction)
- **Non-Linearity**: 1 + 1 + 1 ≠ 3, but = **Fire** (qualitative emergence)
- **Critical Requirement**: Eliminating ANY component stops combustion immediately

## Combustion Process

**Type**: Exothermic oxidation reaction (ΔH < 0)

**Mechanism**:
```
Fuel vaporization → Mixing with O₂ → Oxidation → Heat + Light + Products
```

**Products**:
- CO₂ (Carbon dioxide)
- H₂O (Water vapor)
- Heat (thermal energy)
- Light (electromagnetic radiation)
- Ash/Smoke (incomplete combustion)

**Energy Balance**: Exothermic (releases more energy than absorbed)

**Self-Sustaining**: Yes - positive feedback loop (combustion generates heat → maintains temperature > T_ignition)

def compute_dynamic_asfid(slider_values: dict, base: ASFIDVector,
                           thresholds: dict, fire_on: bool) -> ASFIDVector:
    """
    Compute live ASFID scores from current slider positions.

    The base scores come from the poclet (territorySpace.asfidState).
    We modulate each dimension by how far we are from ignition:

      A - Attractor: peaks when fire is active (system drawn to flame state)
      S - Structure: fuel level drives structural organisation
      F - Flow    : oxygen drives flow intensity
      I - Information: heat level determines signal richness
      D - Dynamics : fire activity drives temporal changes

    All values clamped to [0.0, 1.0].
    """
    fuel_pct  = slider_values.get("fuel",     0.0) / 100.0
    o2_pct    = slider_values.get("oxidizer", 0.0) / 100.0
    heat_pct  = slider_values.get("heat",     0.0) / 100.0   # already 0..1

    fire_f    = 1.0 if fire_on else 0.0
    ignition  = 1.0 if fire_on else max(fuel_pct, o2_pct, heat_pct)

    A = clamp(base.A * (0.3 + 0.7 * fire_f))
    S = clamp(base.S * (0.4 + 0.6 * fuel_pct))
    F = clamp(base.F * (0.2 + 0.8 * o2_pct))
    I = clamp(base.I * (0.3 + 0.7 * heat_pct))
    D = clamp(base.D * (0.1 + 0.9 * ignition))

    return ASFIDVector(A=A, S=S, F=F, I=I, D=D)


def clamp(v, lo=0.0, hi=1.0):
    return max(lo, min(hi, v))


def compute_epistemic_gap(asfid: ASFIDVector, base_asfid: ASFIDVector) -> float:
    """ΔΘ = ‖|Ω_fire⟩ - |M_triangle⟩‖  (simplified: vs base poclet values)"""
    diffs = [
        asfid.A - base_asfid.A,
        asfid.S - base_asfid.S,
        asfid.F - base_asfid.F,
        asfid.I - base_asfid.I,
        asfid.D - base_asfid.D,
    ]
    return math.sqrt(sum(d * d for d in diffs))

class Slider:
    def __init__(self, x, y, w, h, min_val, max_val, init_val,
                 label, unit, threshold=None, threshold_label=""):
        self.rect      = pygame.Rect(x, y, w, h)
        self.min_val   = min_val
        self.max_val   = max_val
        self.value     = float(init_val)
        self.label     = label
        self.unit      = unit
        self.threshold = threshold
        self.threshold_label = threshold_label
        self.dragging  = False
        self.handle_r  = 10

    @property
    def ratio(self):
        return (self.value - self.min_val) / max(self.max_val - self.min_val, 1e-9)

    @property
    def handle_x(self):
        return int(self.rect.x + self.ratio * self.rect.w)

    @property
    def above_threshold(self):
        if self.threshold is None:
            return True
        return self.value >= self.threshold

    def handle_event(self, event):
        if event.type == pygame.MOUSEBUTTONDOWN and event.button == 1:
            hx = self.handle_x
            hy = self.rect.centery
            if math.hypot(event.pos[0] - hx, event.pos[1] - hy) < self.handle_r + 4:
                self.dragging = True
        elif event.type == pygame.MOUSEBUTTONUP and event.button == 1:
            self.dragging = False
        elif event.type == pygame.MOUSEMOTION and self.dragging:
            rel = (event.pos[0] - self.rect.x) / max(self.rect.w, 1)
            self.value = clamp(
                self.min_val + rel * (self.max_val - self.min_val),
                self.min_val, self.max_val
            )

    def draw(self, surf, font_sm, font_xs):
        active = self.above_threshold
        track_color = SLIDER_ACTIVE if active else SLIDER_INACTIVE

        # Track background
        pygame.draw.rect(surf, SLIDER_BG,
                         (self.rect.x, self.rect.centery - 3,
                          self.rect.w, 6), border_radius=3)
        # Filled portion
        fill_w = int(self.ratio * self.rect.w)
        if fill_w > 0:
            pygame.draw.rect(surf, track_color,
                             (self.rect.x, self.rect.centery - 3,
                              fill_w, 6), border_radius=3)

        # Threshold marker
        if self.threshold is not None:
            t_ratio = (self.threshold - self.min_val) / max(self.max_val - self.min_val, 1e-9)
            tx = int(self.rect.x + t_ratio * self.rect.w)
            pygame.draw.line(surf, THRESHOLD_LINE,
                             (tx, self.rect.centery - 12),
                             (tx, self.rect.centery + 12), 2)
            lbl = font_xs.render("min", True, THRESHOLD_LINE)
            surf.blit(lbl, (tx - lbl.get_width() // 2, self.rect.centery + 14))

        # Handle
        handle_color = GOLD if active else (100, 100, 120)
        pygame.draw.circle(surf, handle_color,
                           (self.handle_x, self.rect.centery), self.handle_r)
        pygame.draw.circle(surf, TEXT_COLOR,
                           (self.handle_x, self.rect.centery), self.handle_r, 1)

        # Label + value
        val_str  = f"{self.value:.1f} {self.unit}"
        lbl_surf = font_sm.render(self.label, True, TEXT_COLOR)
        val_surf = font_sm.render(val_str, True, GOLD if active else DIM_COLOR)
        surf.blit(lbl_surf, (self.rect.x, self.rect.y - 22))
        surf.blit(val_surf, (self.rect.right - val_surf.get_width(), self.rect.y - 22))

class Particle:
    def __init__(self, cx, cy, intensity):
        self.x   = cx + random.uniform(-20, 20) * intensity
        self.y   = cy + random.uniform(0, 15)
        self.vx  = random.uniform(-1.5, 1.5) * intensity
        self.vy  = random.uniform(-4, -1.5) * intensity
        self.life = random.uniform(0.4, 1.0)
        self.max_life = self.life
        self.r   = random.uniform(3, 12) * intensity
        self.intensity = intensity

    def update(self, dt):
        self.x    += self.vx
        self.y    += self.vy
        self.vx   *= 0.98
        self.vy   += 0.05   # slight drag
        self.life -= dt
        self.r    = max(0, self.r - 0.15)

    @property
    def alive(self):
        return self.life > 0 and self.r > 0

    def draw(self, surf):
        if not self.alive:
            return
        ratio = self.life / self.max_life
        alpha = int(255 * ratio * self.intensity)
        # Color: white core → orange → red → dark
        if ratio > 0.7:
            color = (255, 255, int(200 * (ratio - 0.7) / 0.3))
        elif ratio > 0.4:
            color = (255, int(120 + 135 * (ratio - 0.4) / 0.3), 20)
        else:
            color = (int(200 * ratio / 0.4), int(40 * ratio / 0.4), 0)

        s = pygame.Surface((int(self.r * 2 + 1), int(self.r * 2 + 1)), pygame.SRCALPHA)
        pygame.draw.circle(s, (*color, alpha), (int(self.r), int(self.r)), int(self.r))
        surf.blit(s, (int(self.x - self.r), int(self.y - self.r)))


class FireSystem:
    def __init__(self, cx, cy):
        self.cx = cx
        self.cy = cy
        self.particles = []
        self.spawn_acc = 0.0
        self.intensity = 0.0   # 0..1, driven externally

    def update(self, dt, active, intensity):
        self.intensity = intensity
        # Spawn
        if active and intensity > 0.01:
            self.spawn_acc += dt * 60 * intensity
            while self.spawn_acc >= 1:
                self.particles.append(Particle(self.cx, self.cy, intensity))
                self.spawn_acc -= 1
        # Update
        for p in self.particles:
            p.update(dt)
        self.particles = [p for p in self.particles if p.alive]

    def draw(self, surf):
        for p in self.particles:
            p.draw(surf)

def draw_fire_triangle(surf, cx, cy, size, sliders, sim_data: PocletSimData,
                       fire_on: bool, font_sm, font_xs):
    """
    Draw the equilateral fire triangle with vertices labelled by component.
    Edges glow when their component is above threshold.
    """
    h = size * math.sqrt(3) / 2
    # Vertices: top=Heat, bottom-left=Fuel, bottom-right=Oxidizer
    pts = [
        (cx,              cy - h * 2 / 3),     # top
        (cx - size / 2,   cy + h / 3),          # bottom-left
        (cx + size / 2,   cy + h / 3),          # bottom-right
    ]

    # Map component labels to vertices (order from parser)
    labels  = [c.label for c in sim_data.components]
    # Default vertex order: Fuel=0, Oxidizer=1, Heat=2
    # We map: Heat→top, Fuel→bottom-left, Oxidizer→bottom-right
    vertex_map = {}
    for i, c in enumerate(sim_data.components):
        lbl = c.label.lower()
        if "heat" in lbl:
            vertex_map[i] = 0   # top
        elif "fuel" in lbl or "combustible" in lbl:
            vertex_map[i] = 1   # bottom-left
        else:
            vertex_map[i] = 2   # bottom-right

    # Edge colors: lit when component above threshold
    def comp_active(idx):
        s = list(sliders.values())[idx]
        return s.above_threshold

    edge_colors = []
    for i in range(len(sim_data.components)):
        edge_colors.append(FIRE_ACTIVE if comp_active(i) else (50, 50, 70))

    # Draw edges (triangle sides)
    for i in range(3):
        j = (i + 1) % 3
        col = FIRE_ACTIVE if fire_on else (50, 50, 70)
        # Glow
        if fire_on:
            for gw in range(6, 0, -2):
                alpha_col = (col[0], col[1], col[2])
                pygame.draw.line(surf, alpha_col, pts[i], pts[j], gw)
        pygame.draw.line(surf, col, pts[i], pts[j], 2)

    # Draw vertices
    for i, c in enumerate(sim_data.components):
        vi = vertex_map.get(i, i)
        px, py = pts[vi]
        active = comp_active(i)
        vcol = FIRE_ACTIVE if active else (80, 80, 100)
        pygame.draw.circle(surf, vcol, (int(px), int(py)), 8)
        pygame.draw.circle(surf, TEXT_COLOR, (int(px), int(py)), 8, 1)

        # Label
        lbl_lines = c.label.split("(")
        lbl_short = lbl_lines[0].strip()
        lbl_surf = font_sm.render(lbl_short, True, TEXT_COLOR if active else DIM_COLOR)
        # Position label outside vertex
        dx = px - cx
        dy = py - cy
        lx = px + dx * 0.35 - lbl_surf.get_width() / 2
        ly = py + dy * 0.35 - lbl_surf.get_height() / 2
        surf.blit(lbl_surf, (lx, ly))

    # Center: "FIRE" label
    fire_col = GOLD if fire_on else (60, 60, 80)
    txt = font_sm.render("FIRE" if fire_on else "—", True, fire_col)
    surf.blit(txt, (cx - txt.get_width() // 2, cy - txt.get_height() // 2))

def draw_panel(surf, rect, title, font_md, color=PANEL_COLOR, border=BORDER_COLOR):

    pygame.draw.rect(surf, border, rect, 1, border_radius=8)

        surf.blit(t, (rect.x + PANEL_PAD, rect.y + PANEL_PAD))

def draw_bar(surf, x, y, w, h, value, max_val, color, bg_color=SLIDER_BG, label="", font=None):
    pygame.draw.rect(surf, bg_color, (x, y, w, h), border_radius=3)
    fill = int(clamp(value / max(max_val, 1e-9)) * w)
    if fill > 0:
        pygame.draw.rect(surf, color, (x, y, fill, h), border_radius=3)
    if label and font:
        lbl = font.render(label, True, TEXT_COLOR)
        surf.blit(lbl, (x, y - lbl.get_height() - 2))

def main():
    # --- Load poclet via PocletRepository (auto-discovers repo root) ---
    try:
        if POCLET_PATH_OVERRIDE:
            print(f"[TSCG] Loading poclet from explicit path: {POCLET_PATH_OVERRIDE}")
            sim_data = PocletParser(POCLET_PATH_OVERRIDE).parse()
        else:
            repo = PocletRepository()
            print(f"[TSCG] Repository root : {repo.root}")
            print(f"[TSCG] Poclets found   : {repo.list()}")
            sim_data = repo.load("FireTriangle")
    except FileNotFoundError as e:
        print(f"[ERROR] {e}")
        print("  Tip: run from anywhere inside the tscg repository,")
        print("  or pass an explicit path: python fire_triangle_sim.py path/to/M0_FireTriangle.jsonld")
        sys.exit(1)
    except Exception as e:
        print(f"[ERROR] Failed to parse poclet: {e}")
        raise

    print(f"[TSCG] Poclet loaded: '{sim_data.label}'")
    print(f"[TSCG] Components   : {[c.label for c in sim_data.components]}")
    print(f"[TSCG] ASFID base   : {sim_data.asfid_base}")
    print(f"[TSCG] REVOI        : {sim_data.revoi}")

    # --- Pygame init ---
    pygame.init()
    screen  = pygame.display.set_mode((WINDOW_W, WINDOW_H))
    pygame.display.set_caption(f"TSCG Simulation · {sim_data.label}")
    clock   = pygame.time.Clock()

    font_lg = pygame.font.SysFont("segoeui", 20, bold=True)
    font_md = pygame.font.SysFont("segoeui", 15, bold=True)
    font_sm = pygame.font.SysFont("segoeui", 13)
    font_xs = pygame.font.SysFont("segoeui", 11)

    # --- Build sliders from poclet components ---
    # We position them in the left panel
    lp_x  = PANEL_PAD
    slider_keys = []
    sliders = {}

    # Physical defaults when poclet constraints are textual (no numeric value)
    COMPONENT_DEFAULTS = {
        "heat":     {"min": 0.0,  "max": 1000.0, "unit": "°C",   "init": 50.0,
                     "threshold": 233.0, "threshold_label": "T_ignition paper ~233°C"},
        "fuel":     {"min": 0.0,  "max": 100.0,  "unit": "%",    "init": 20.0,
                     "threshold": 20.0,  "threshold_label": "Above LEL (~20%)"},
        "oxidizer": {"min": 0.0,  "max": 100.0,  "unit": "% O₂", "init": 10.0,
                     "threshold": 16.0,  "threshold_label": "> 16% for sustained combustion"},
    }

    for i, comp in enumerate(sim_data.components):
        lbl_lower = comp.label.lower()
        # Detect component type
        if "heat" in lbl_lower:
            ctype = "heat"
        elif "fuel" in lbl_lower or "combustible" in lbl_lower:
            ctype = "fuel"
        else:
            ctype = "oxidizer"

        defaults = COMPONENT_DEFAULTS[ctype]
        # Use poclet threshold if parser extracted one, else physical default
        threshold       = comp.threshold if comp.threshold is not None else defaults["threshold"]
        threshold_label = comp.threshold_label or defaults["threshold_label"]

        key = ctype
        slider_keys.append(key)
        sliders[key] = Slider(
            x=lp_x + PANEL_PAD,
            y=120 + i * 90,
            w=LEFT_PANEL_W - PANEL_PAD * 4,
            h=20,
            min_val=defaults["min"],
            max_val=defaults["max"],
            init_val=defaults["init"],
            label=comp.label.split("(")[0].strip(),
            unit=defaults["unit"],
            threshold=threshold,
            threshold_label=threshold_label,
        )

    # Fire system (center panel)
    mid_cx = LEFT_PANEL_W + MID_W // 2
    mid_cy = WINDOW_H // 2 + 20
    fire_sys = FireSystem(mid_cx, mid_cy + 50)

    # --- Main loop ---
    running = True
    while running:
        dt = clock.tick(FPS) / 1000.0

        # Events
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                running = False
            if event.type == pygame.KEYDOWN and event.key == pygame.K_ESCAPE:
                running = False
            for s in sliders.values():
                s.handle_event(event)

        # Fire logic: all components above threshold?
        fire_on = all(s.above_threshold for s in sliders.values())

        # Intensity: how far above thresholds (0..1)
        if fire_on:
            scores = []
            for s in sliders.values():
                r = (s.value - (s.threshold or s.min_val)) / max(s.max_val - (s.threshold or s.min_val), 1e-9)
                scores.append(clamp(r))
            intensity = sum(scores) / len(scores)
        else:
            intensity = 0.0

        # Dynamic ASFID
        slider_norm = {}
        for key, s in sliders.items():
            if "heat" in key:
                slider_norm[key] = s.value / s.max_val * 100.0
            else:
                slider_norm[key] = s.value
        # Use simpler normalized values
        sv = {}
        for key, s in sliders.items():
            sv[key] = (s.value - s.min_val) / max(s.max_val - s.min_val, 1e-9) * 100.0

        asfid_live = compute_dynamic_asfid(sv, sim_data.asfid_base, {}, fire_on)

# rag_simulation_kb.py
import json
import numpy as np
from sentence_transformers import SentenceTransformer
import chromadb
from chromadb.utils import embedding_functions

class SimulationRAG:
    def __init__(self, persist_directory="./simulation_rag_db"):
        # Initialisation du modèle d'embeddings
        self.embedding_model = SentenceTransformer('all-MiniLM-L6-v2')
        
        # Initialisation ChromaDB
        self.chroma_client = chromadb.PersistentClient(path=persist_directory)
        
        # Collections pour différents types de patterns
        self.structural_collection = self.chroma_client.get_or_create_collection(
            name="structural_patterns",
            embedding_function=embedding_functions.SentenceTransformerEmbeddingFunction()
        )
        
        self.dynamic_collection = self.chroma_client.get_or_create_collection(
            name="dynamic_patterns"
        )
        
        self.ui_collection = self.chroma_client.get_or_create_collection(
            name="ui_patterns"
        )
        
        self.asfid_collection = self.chroma_client.get_or_create_collection(
            name="asfid_patterns"
        )
        
    def add_simulation_patterns(self, simulation_name, patterns):
        """Ajoute tous les patterns d'une simulation à la base RAG"""
        
        # Créer un document composite pour le retrieval
        composite_text = f"""
        Simulation: {simulation_name}
        Type: {patterns['structural_pattern']['pattern_type']}
        Composants: {patterns['structural_pattern']['signature']['components']}
        ASFID dimensions: {list(patterns['asfid_revoi_pattern']['asfid']['dimensions'])}
        """
        
        # Embedding pour recherche sémantique
        embedding = self.embedding_model.encode(composite_text)
        
        # Stockage dans les différentes collections
        collection_map = {
            self.structural_collection: patterns['structural_pattern'],
            self.dynamic_collection: patterns['dynamic_pattern'],
            self.ui_collection: patterns['interaction_pattern'],
            self.asfid_collection: patterns['asfid_revoi_pattern']
        }
        
        for collection, pattern_data in collection_map.items():
            collection.add(
                documents=[json.dumps(pattern_data)],
                metadatas=[{
                    "simulation": simulation_name,
                    "pattern_type": pattern_data.get("pattern_type", "unknown"),
                    "complexity": patterns.get("complexity", 1)
                }],
                ids=[f"{simulation_name}_{collection.name}"]
            )
    
    def find_similar_patterns(self, query, pattern_type=None, n_results=3):
        """Recherche des patterns similaires à une requête"""
        
        collection = self._get_collection(pattern_type)
        results = collection.query(
            query_texts=[query],
            n_results=n_results
        )
        
        return results
    
    def generate_context_for_new_simulation(self, ontology_description):
        """Génère un contexte enrichi pour la création d'une nouvelle simulation"""
        
        # Recherche de patterns similaires
        similar_structural = self.find_similar_patterns(
            ontology_description, 
            pattern_type="structural"
        )
        
        similar_dynamic = self.find_similar_patterns(
            ontology_description,
            pattern_type="dynamic"
        )
        
        # Construction du contexte
        context = {
            "similar_structural_patterns": similar_structural,
            "similar_dynamic_patterns": similar_dynamic,
            "suggested_ui_pattern": self._suggest_ui_pattern(ontology_description),
            "asfid_mapping_suggestions": self._suggest_asfid_mappings(ontology_description),
            "code_templates": self._get_relevant_templates(ontology_description)
        }
        
        return context

    def extract_structural_pattern(self):
        """Extrait la structure du système de l'ontologie"""
        
        # Chercher les composants dans l'ontologie
        components = []
        synergy = {}
        
        # Parcourir le graphe JSON-LD
        for item in self.ontology.get('@graph', []):
            # Chercher les composants
            if 'm0:fire_triangle:components' in item:
                for comp in item['m0:fire_triangle:components']:
                    if isinstance(comp, dict):
                        components.append({
                            'name': comp.get('rdfs:label', 'Unknown'),
                            'role': comp.get('m0:fire_triangle:role', ''),
                            'function': comp.get('m0:fire_triangle:function', ''),
                            'asfid_contrib': comp.get('m0:fire_triangle:asfidContribution', {})
                        })
            
            # Chercher le principe de synergie
            if 'm0:fire_triangle:synergyPrinciple' in item:
                synergy = item['m0:fire_triangle:synergyPrinciple']
        
        # Analyser le code pour trouver la correspondance
        class_names = self._extract_classes_from_code()
        slider_count = len([c for c in class_names if 'Slider' in c])
        
        pattern = {
            "pattern_type": "ternary_synergistic_system",

            },
            "code_mapping": {

    def extract_dynamic_pattern(self):
        """Extrait le comportement dynamique du code"""
        
        # Trouver la fonction de mise à jour
        update_func = self._find_function('update')
        
        # Trouver la boucle principale
        main_loop = self._find_main_loop()
        
        # Extraire les équations dynamiques
        equations = self._extract_dynamic_equations()
        
        pattern = {
            "pattern_type": "threshold_crossing_dynamics",
            "state_machine": self._extract_state_machine(),
            "continuous_dynamics": {
                "equations": equations,
                "time_step": "dt parameter in update()",
                "particle_system": self._analyze_particle_system()
            },
            "feedback_loops": self._identify_feedback_loops(),
            "code_location": {
                "update_function": update_func,
                "main_loop": main_loop
            }
        }
        
        return pattern
    
    def _find_function(self, func_name):
        """Trouve une fonction dans l'AST"""
        for node in ast.walk(self.code_ast):
            if isinstance(node, ast.FunctionDef) and func_name in node.name:
                return {
                    'name': node.name,
                    'line': node.lineno,
                    'params': [arg.arg for arg in node.args.args]
                }
        return None
    
    def _extract_dynamic_equations(self):
        """Extrait les équations de modulation ASFID"""
        equations = {}
        
        # Chercher compute_dynamic_asfid
        for node in ast.walk(self.code_ast):
            if isinstance(node, ast.FunctionDef) and node.name == 'compute_dynamic_asfid':
                for child in ast.walk(node):
                    if isinstance(child, ast.Assign):
                        # Extraire les équations comme 'A = clamp(base.A * (0.3 + 0.7 * fire_f))'
                        if isinstance(child.targets[0], ast.Name):
                            var_name = child.targets[0].id
                            if var_name in ['A', 'S', 'F', 'I', 'D']:
                                equations[var_name] = ast.unparse(child.value)
        
        return equations
    
    def _extract_state_machine(self):
        """Extrait la machine à états du code"""
        # Chercher la condition fire_on = all(s.above_threshold)
        state_logic = []
        
        for node in ast.walk(self.code_ast):
            if isinstance(node, ast.Assign):
                for target in node.targets:
                    if isinstance(target, ast.Name) and target.id == 'fire_on':
                        state_logic.append({
                            'state_var': 'fire_on',
                            'condition': ast.unparse(node.value),
                            'line': node.lineno
                        })
        
        return {
            "states": ["INACTIVE", "ACTIVE"],
            "transition_condition": state_logic[0]['condition'] if state_logic else "unknown"
        }

        slider_class = self._find_class('Slider')
        
        # Analyser le système de particules
        particle_system = self._find_class('FireSystem')
        
        # Analyser la disposition
        layout = self._extract_layout()
        
        pattern = {
            "pattern_type": "direct_manipulation_interface",
            "components": {
                "sliders": self._analyze_slider_class(slider_class),
                "visualization": {
                    "triangle": self._find_triangle_drawing(),
                    "particle_system": self._analyze_particle_system()
                }
            },
            "layout": layout,
            "color_scheme": self._extract_color_scheme(),
            "event_handling": self._extract_event_handling()
        }
        
        return pattern
    
    def _find_class(self, class_name):
        """Trouve une classe dans l'AST"""
        for node in ast.walk(self.code_ast):
            if isinstance(node, ast.ClassDef) and class_name in node.name:
                methods = []
                for child in node.body:
                    if isinstance(child, ast.FunctionDef):
                        methods.append({
                            'name': child.name,
                            'params': [arg.arg for arg in child.args.args],
                            'line': child.lineno
                        })
                
                return {
                    'name': node.name,
                    'methods': methods,
                    'line': node.lineno
                }
        return None
    
    def _analyze_slider_class(self, slider_class):
        """Analyse en détail la classe Slider"""
        if not slider_class:
            return {}
        
        properties = []
        events = []
        
        # Chercher dans l'AST les attributs et méthodes
        for node in ast.walk(self.code_ast):
            if isinstance(node, ast.ClassDef) and node.name == slider_class['name']:
                for child in node.body:
                    if isinstance(child, ast.FunctionDef):
                        if 'event' in child.name or 'handle' in child.name:
                            events.append(child.name)
                    elif isinstance(child, ast.AnnAssign):
                        if isinstance(child.target, ast.Name):
                            properties.append(child.target.id)
        
        return {
            "properties": properties,
            "events": events,
            "threshold_logic": "above_threshold property",
            "visual_feedback": ["color change", "threshold marker", "value display"]
        }
    
    def _extract_layout(self):
        """Extrait la disposition des panneaux"""
        layout = {
            "left_panel": "controls",
            "center_panel": "visualization", 
            "right_panel": "analytics"
        }
        
        # Chercher les constantes de dimensions
        for node in ast.walk(self.code_ast):
            if isinstance(node, ast.Assign):
                for target in node.targets:
                    if isinstance(target, ast.Name):
                        if 'LEFT_PANEL_W' in target.id:
                            layout['left_width'] = self._get_constant_value(node.value)
                        elif 'RIGHT_PANEL_W' in target.id:
                            layout['right_width'] = self._get_constant_value(node.value)
        
        return layout
    
    def _get_constant_value(self, node):
        """Extrait la valeur d'une constante"""
        if isinstance(node, ast.Constant):
            return node.value
        return None

        for item in self.ontology.get('@graph', []):
            if 'rdfs:label' in item:
                desc_parts.append(f"System: {item['rdfs:label']}")
            if 'rdfs:comment' in item:
                desc_parts.append(f"Description: {item['rdfs:comment']}")
        
        # Ajouter les composants
        for item in self.ontology.get('@graph', []):
            if 'm0:fire_triangle:components' in item:
                for comp in item['m0:fire_triangle:components']:
                    if isinstance(comp, dict):
                        desc_parts.append(f"Component: {comp.get('rdfs:label', '')}")
        
        return " | ".join(desc_parts)

    def _assess_complexity(self):
        """Évalue la complexité de la simulation"""
        score = 1  # Base
        
        # Nombre de composants
        comp_count = 0
        for item in self.ontology.get('@graph', []):
            if 'm0:fire_triangle:components' in item:
                comp_count = len(item['m0:fire_triangle:components'])
        score += comp_count * 0.5
        
        # Nombre de classes dans le code
        class_count = len(self._extract_classes_from_code())
        score += class_count * 0.3
        
        return min(round(score, 1), 5)  # Max 5

        if domain:

    def save_to_file(self, output_path):
        """Sauvegarde les patterns dans un fichier JSON"""
        
        # Convertir en format sérialisable
        serializable = self._make_serializable(self.patterns)
        
        output_path = Path(output_path)
        output_path.parent.mkdir(parents=True, exist_ok=True)
        
        with open(output_path, 'w', encoding='utf-8') as f:
            json.dump(serializable, f, indent=2, ensure_ascii=False)
        
        print(f"💾 Patterns sauvegardés dans {output_path}")
        
        return output_path
    
    def _make_serializable(self, obj):
        """Convertit les objets non sérialisables en types de base"""
        if isinstance(obj, dict):
            return {k: self._make_serializable(v) for k, v in obj.items()}
        elif isinstance(obj, list):
            return [self._make_serializable(v) for v in obj]
        elif hasattr(obj, '__dict__'):
            return self._make_serializable(obj.__dict__)
        else:
            return obj

class TSCGPatternExtractor:
    # ... (code précédent)
    
    def save_as_m1_extension(self, output_path):
        """Sauvegarde les patterns comme extension M1"""
        
        m1_graph = {
            "@context": {
                "dcterms": "http://purl.org/dc/terms/",
                "owl": "http://www.w3.org/2002/07/owl#",
                "rdf": "http://www.w3.org/1999/02/22-rdf-syntax-ns#",
                "rdfs": "http://www.w3.org/2000/01/rdf-schema#",
                "m2": "https://raw.githubusercontent.com/Echopraxium/tscg/main/ontology/M2_MetaConcepts.jsonld#",
                "m1:sim": "https://raw.githubusercontent.com/Echopraxium/tscg/main/ontology/M1_extensions/M1_SimulationGame.jsonld#"
            },
            "@graph": []
        }
        
        # Créer l'instance de pattern structurel
        structural_instance = {
            "@id": f"m1:sim:{self.patterns['metadata']['simulation_name'].replace(' ', '')}Structural",
            "@type": "owl:NamedIndividual",
            "rdf:type": "m1:sim:StructuralPattern",
            "rdfs:label": f"{self.patterns['metadata']['simulation_name']} Structural Pattern",
            "m1:sim:signature": self.patterns['structural']['signature'],
            "m1:sim:mappingRules": self.patterns['structural'].get('mapping_rules', []),
            "m2:asfidState": self.patterns['asfid_revoi']['asfid']['base_values'],
            "m1:sim:sourcePoclet": str(self.ontology_path)
        }
        m1_graph["@graph"].append(structural_instance)
        
        # Créer l'instance de pattern dynamique
        dynamic_instance = {
            "@id": f"m1:sim:{self.patterns['metadata']['simulation_name'].replace(' ', '')}Dynamic",
            "@type": "owl:NamedIndividual",
            "rdf:type": "m1:sim:DynamicPattern",
            "rdfs:label": f"{self.patterns['metadata']['simulation_name']} Dynamic Pattern",
            "m1:sim:stateMachine": self.patterns['dynamic']['state_machine'],
            "m1:sim:continuousDynamics": self.patterns['dynamic']['continuous_dynamics'],
            "m2:asfidState": {
                "D": self.patterns['asfid_revoi']['asfid']['base_values'].get('D', 0),
                "F": self.patterns['asfid_revoi']['asfid']['base_values'].get('F', 0)
            }
        }
        m1_graph["@graph"].append(dynamic_instance)
        
        # Sauvegarder
        with open(output_path, 'w', encoding='utf-8') as f:
            json.dump(m1_graph, f, indent=2, ensure_ascii=False)
        
        return output_path
Avantages de cette approche
Cohérence architecturale : Les patterns s'intègrent parfaitement dans la hiérarchie M3→M2→M1→M0

Réutilisabilité : Les patterns M1 peuvent être référencés par plusieurs poclets M0

Traçabilité : Chaque pattern garde la trace de ses instances (Fire Triangle, RGB, etc.)

Évolutivité : Nouveaux patterns = nouvelles instances dans M1_SimulationGame.jsonld

Interopérabilité : Les patterns peuvent être utilisés par d'autres extensions M1 (chimie, physique, etc.)

Utilisation dans le pipeline RAG
python
class RAGWithM1:
    def __init__(self, m1_simulation_path):
        self.m1_patterns = self._load_m1_patterns(m1_simulation_path)
        
    def _load_m1_patterns(self, path):
        with open(path, 'r') as f:
            return json.load(f)
    
    def find_patterns_for_new_system(self, system_description):
        """Trouve les patterns M1 pertinents"""
        matching_patterns = []
        
        for pattern in self.m1_patterns['@graph']:
            if pattern.get('@type') == 'owl:NamedIndividual':
                # Calculer similarité sémantique
                similarity = self._compute_similarity(
                    system_description,
                    pattern.get('rdfs:label', '') + ' ' + pattern.get('rdfs:comment', '')
                )
                
                if similarity > 0.6:
                    matching_patterns.append({
                        'pattern': pattern,
                        'similarity': similarity
                    })
        
        return sorted(matching_patterns, key=lambda x: x['similarity'], reverse=True)
